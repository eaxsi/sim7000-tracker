# Development environment
- Create a file called `arduino_secrets.h`

The file should have the following structure
```
#define SECRET_BROKER_HOST ""
#define SECRET_BROKER_USER ""
#define SECRET_BROKER_PW ""

#define HTTP_OTA_URL ""
```

Library versions

- PubSubClient: v2.8.0
